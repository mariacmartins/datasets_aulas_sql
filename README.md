## 🛠️ O que você vai praticar com estes dados?

Ao longo das aulas, utilizaremos estes arquivos para dominar o uso do **SQL**, cobrindo tópicos como:

* **Consultas Básicas e Filtros:** Uso de `SELECT`, `LIMIT`, `DISTINCT` e apelidos (`AS`).
* **Regras de Negócio e Refinamento:** Filtros com `WHERE` utilizando operadores aritméticos, lógicos (`AND`, `OR`, `NOT`) e de comparação (`>`, `<`, `!=`).
* **Entendimento sobre condicionais e cláusulas de agregação e agrupamento:** Praticar agrupamento de dados, como aplicar regras condicionais aos dados, como ordenar dados, etc.
* **Cruzamento de dados:** Entender como o SQL resolve problemas de cruzamento entre tabelas, através uso de JOINs.
* **Subqueries vs CTEs:** Quando utilizar cada abordagem.
